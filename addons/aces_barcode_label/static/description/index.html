Odoo 8 Product Label Report Extension

Product custom label design. User can change the sequence of fields to display in label and can change font-size, line margin and color as user requires.

How to use:

  • User has to select product(s) from a list or form view, and click on dynamic product label from print button. one wizard will open. User has to fill appropriate data in wizard and click on print.
  • select product(s) from list view or form view.




  • By clicking on "Dynamic Product Label" menu from print button wizard will popup, and user has to fill appropriate data.
  • In, "Products" tab user will get product list which user has selected from list or form view or can add product(s) in list .


  • In, "Fields" tab user can select fields of product which user need to print in label.
